Expense Tracker REST API

This is a backend REST API. You can interact with it using tools like curl, Postman, or your browser for GET endpoints.

  • List expenses: GET /api/expenses
  • Add expense: POST /api/expenses (JSON: {"description":..., "amount":...})
  • Total expenses: GET /api/expenses/total
  • Clear expenses: DELETE /api/expenses

Example: curl -X POST -H "Content-Type: application/json" -d '{"description":"Coffee","amount":3.5}' http://localhost:8080/api/expenses
